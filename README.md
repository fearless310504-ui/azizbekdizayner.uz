# Azizbek's Portfolio Website

A modern, animated portfolio website for graphic designer Azizbek Abdunazarov.

## Features

- Responsive design with Tailwind CSS
- Smooth animations with AOS and GSAP
- Particle background effects
- Contact form with backend email sending
- Music player
- Video background

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables for email:
   Create a `.env` file in the root directory:
   ```
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASS=your-app-password
   ```

3. Run the server:
   ```bash
   npm start
   ```

4. Open http://localhost:3000 in your browser.

## Technologies

- Frontend: HTML, Tailwind CSS, JavaScript, AOS, GSAP, Particles.js
- Backend: Node.js, Express, Nodemailer
